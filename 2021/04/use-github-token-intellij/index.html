인텔리제이에서 Github Token 사용하기

2021-04-15
⏱ 1 min.
intellij, tutorial

평소 터미널로 버전관리를 했는데, 얼마전 코드리뷰를 받는 과정에서 Github Token을 발급하여 인텔리제이에 적용할 일이 발생했다.

리뷰어께서 CodeWithMe 기능을 이용해 버전관리 관련 도움을 주려고 하는 과정에서 인텔리제이에 Github Token이 안되있어서 매번 웹으로 로그인했기 때문이다.

Github에서 이중인증을 사용한다면 인텔리제이 뿐만 아니라 버전관리를 하는 모든 클라이언트에 Token을 적용해야 한다. 그래야 매번 사용할때마다 로그인을 할 필요가 없다. 인텔리제이에서는 아래와 같은 경고창을 마주하게 된다.

여기서 [Use Token] 을 클릭하면 아래와 같은 창이 출력된다.

여기서 [Generate] 버튼을 클릭하면, 브라우저에서 Github Token을 생성하는 창이 열린다.

해당 토큰의 Scope를 인텔리제이에서 요구하는걸 자동으로 선택되므로 다른걸 만지지 않고 이대로 토큰을 생성했다.

이미지에선 토큰주소를 가려뒀지만, 여기에서 발생한 토큰을 클립보드에 복사해서 인텔리제이에 붙히면 마무리가 된다.

토큰주소를 만약 잃어버렸다면 Github의 Personal aceess tokens에서 방금 생성한 토큰을 클릭하여 [Regenerate token] 을 클릭하면 다시 생성된다.
